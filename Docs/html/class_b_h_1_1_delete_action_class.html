Represents the action of deleting one or more targets.

Used to save and restore dominos.

Dictionary<int, Selectable> BH.DeleteActionClass.GetUpdatedInstances ( )

Getter for the mapping between old Selectable instance IDs and new Selectable objects.

Returns
A Dictionary<int,Selectable> mapping old instance IDs of the deleted targets to the new Selectable instances of the targets. Empty if new Selectables weren't spawned, i.e. undo() hasn't been called yet.
