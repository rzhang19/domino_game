◆ AttemptUndo()

bool BH.ActionHistory.AttemptUndo ( )

Attempts an undo by popping the last action off the stack.

If the action was a delete, undoing it has changed some Selectable IDs by respawning them, so update all relevant actions to use the new Selectable instances.

Returns
True if the undo succeeded. Note: If it returns false, the action history may be corrupted. This should never happen though.
