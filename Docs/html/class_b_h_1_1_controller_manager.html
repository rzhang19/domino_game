Detailed Description

Manages the "mode" of the player during gameplay.

The player can either be in "build mode" or "spectator mode". From either of those modes, the player can also toggle between "free-fly" controls or "fixed-camera" controls.

A client can make calls through ControllerManager to change the mode and controls of the player.
