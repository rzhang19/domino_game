Detailed Description

Implements an interface for UI objects to interact with a BuildModeController.

This class is an adapter for BuildModeController with some additional functionalities. This class exists because Unity's UI objects are limited to calling no-argument and one-argument functions.

See also
BH.BuildModeController, UnityEngine.MonoBehaviour
